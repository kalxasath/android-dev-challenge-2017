# Try Catch Structure
This is a typical try catch structure with just log any errors
```java
    try {
		// your code to try goes here
    } catch (Exception e) {
        Log.e(TAG, "Failed to ....");
        e.printStackTrace();
        return null;
    }

    return <returnType>;
```
